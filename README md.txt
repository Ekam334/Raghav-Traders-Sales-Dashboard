Raghav Traders Sales Dashboard (Power BI)

An interactive Power BI dashboard built using two CSV datasets to visualize sales, profit, quantity, and customer trends for Raghav Traders. The dashboard provides insights by state, quarter, category, payment mode, and month, helping identify high-performing segments.

Features:
1. State-wise sales and revenue analysis
2. Monthly profit trends
3. Category and sub-category performance
4. Customer-wise revenue
5. Payment mode distribution
6. Dynamic slicers: State, Quarter (Q1â€“Q4)
7. KPI cards for Total Sales, Profit, and Quantity

Tech Stack:
Power BI Desktop, Power Query, MS Excel

How to use:
1. Clone the repo
2. Open Dashboard.pbi in PowerBI Desktop
3. Ensure CSV files are in the /data folder
4. Refresh the report
